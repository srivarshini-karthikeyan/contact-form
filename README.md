## 📋 What is this?

A clean, minimal **Contact Form** built with pure `HTML`, `CSS`, and `JavaScript`. It collects a user's name, phone number, and email — with client-side validation and a reset option. CSS-first at **81.9%** of the codebase, the project prioritises a polished visual presentation over heavy scripting.

Zero dependencies. Zero frameworks. Instant deployment via GitHub Pages.

---

## 📝 Form Fields

<div align="center">

| Field | Type | Validation |
|:---|:---:|:---|
| 👤 **Name** | Text | Required, non-empty |
| 📱 **Phone Number** | Tel | Required, valid phone format |
| 📧 **Email Address** | Email | Required, valid email format |
